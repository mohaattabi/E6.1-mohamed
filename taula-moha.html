Atributo Descripción Explicación Ejemplo
background-color Establece el color de fondo de un elemento. El fondo de un elemento es el tamaño total del elemento, incluyendo las protecciones y la frontera (pero no el margen). body { background-color: yellow; } h1 { background-color: #00ff00; } p { background-color: rgb(255,0,255); }
background-image Una imagen de fondo El atributo especifica el fondo de un documento. h1>Hello world!/h1 p>a href="http://www.w3schools.com">Visit W3Schools.com!a/p /body /html
background-repeat se establece si / cómo se repetirá una imagen de fondo. Por defecto, el fondo-imagen se repite tanto vertical como horizontalmente. body { background-image: url("paper.gif"); background-repeat: repeat-y; }
font-size especifica el tamaño del texto dentro de un elemento. El atributo de tamaño de no es compatible con HTML5. Usar CSS lugar. font size="6">This is some text!/font>
font-weight establece cómo se deben mostrar caracteres gruesos o finos en el texto. Establecer diferente peso de la fuente de tres párrafos p.normal { font-weight: normal; } p.thick { font-weight: bold; } p.thicker { font-weight: 900; }
font-family especifica la fuente para un elemento. puede contener varios nombres de fuente como un sistema de "retroceso". Si el navegador no es compatible con la primera fuente, que trata la siguiente fuente. p { font-family: "Times New Roman", Georgia, Serif; }

 p align="right">This is some text in a paragraph./p>
text-transform La propiedad text-transform controla la capitalización de texto. Transformar texto en diferentes p> elementos p.uppercase { text-transform: uppercase; } p.lowercase { text-transform: lowercase; } p.capitalize { text-transform: capitalize; }
border especifica qué tipo de frontera que se vea puede tener de uno a cuatro valores (por el borde superior, borde derecho, borde inferior, y la frontera de la izquierda). p.dotted {border-style: dotted;}
list-style-type especifica el tipo de marcador de la lista de ítems en una lista. Establece algunas diferentes estilos de lista ul.circle {list-style-type: circle;}
text-decoration especifica la decoración añadida al texto. Establezca la decoración de texto para h1>, h2> y h3> elementos h1 { text-decoration: overline; } h2 { text-decoration: line-through; } h3 { text-decoration: underline; }
padding establece todas las propiedades de relleno en una declaración. Establecer el relleno de un elemento p> p {padding: 2cm 4cm 3cm 4cm;}
margin establece todas las propiedades de margen en una declaración. Ajuste todos los cuatro márgenes de un elemento p> p {margin: 2cm 4cm 3cm 4cm;}
