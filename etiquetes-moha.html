Etiquetas HTML

Etiquetas de Texto

"U": Esta etiqueta sirve para que se subraye el texto, apareciendo una línea por debajo del mismo.
"BLOCKQUOTE":Esta etiqueta es bastante útil, porque crea un margen tanto a la izquierda como la derecha, lo que produce un efecto como si fuera un fragmento extraído de otro lugar o que sirva para comentar la información anterior.
"TT": Texto de Teletipo. Que crea un estilo como el de las antiguas máquinas de teletipo, que transmitían información con caracteres.
"strike": Esta etiqueta no es muy conocida y tampoco muy utilizada. Sirve para que el texto aparezca tachado, es decir, con una línea por encima del texto.
"CODE": Sirve para escribir un código fuente de un lenguaje de programación. Simplemente coloca una fuente de un tipo como courier, en el que todas las letras tienen la misma anchura.
"H1":sirve para destacar un texto, pero no las podemos utilizar en cualquier caso, porque un encabezamiento siempre indica un titular o la apertura de una nueva sección con un título.
"I":Las itálicas o cursivas, que sirven para hacer que el texto se incline a un lado.
"CENTER":Permite centrar todo el texto del párrafo.
"CITE":Sirve para colocar texto de cita, texto dicho por otra persona. Esta etiqueta generalmente pone el texto en cursiva, pero otros navegadores podrían interpretarla de otras maneras.
"P": Sirve para delimitar un párrafo. Inserta una línea en blanco antes del texto.

Etiquetas de Imagenes

"SRC":Este atributo es obligatorio e indica el nombre del archivo de imagen (entre comillas) o la URL desde la que se va a obtener la imagen.
"height": Este atributo indica el alto de la imagen.

Etiquetas de Enlaces

" _self": Abre el documento vinculado en el mismo marco o ventana que el vínculo.

Etiquetas para Ordenar

"div": se utiliza para definir una sección dentro del documento.
"OL": se utiliza para crear una lista ordenada.
"Li": nos define los objetos de la lista.
"UL": se utiliza para crear una lista desordenada.

Etiquetas para Listar

"dl": nos indica que dentro de ella, entre ella y su cierre, va a ir una definición.
"dt": sirve para entendernos mejor, dentro de "dt" iría el título de la definición.
"dd": nos dice que dentro de ésta irá la definición.
